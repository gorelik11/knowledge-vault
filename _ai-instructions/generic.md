# Knowledge Base Protocol — Generic AI Tool

This vault at ~/Knowledge/ contains structured knowledge files organized by project.

## How to use:
1. Each project has a section (folder) with an index.md
2. Read your project's index.md to see available topics
3. Load topic files on demand when relevant to the current task
4. Never load everything at once — selective retrieval only
5. brain.md is the master map — load only for reorganization
6. _infrastructure/, _agents/, _shared/ contain cross-project knowledge
