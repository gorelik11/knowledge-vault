# Rogachev — Practical Rules for MIDI Composition

## Seven Types of Fundamental Tone Movement

All jazz harmony based on **7 интервальных направлений** movement of roots:

| Type | Interval | Example (from C) | Function |
|------|----------|------------------|----------|
| 1 | Кварто-квинтовое (P4↑/P5↓) | C → F or C → G | Strongest (ii-V-I) |
| 2 | Нисходящее полутоновое (m2↓) | C → B | Tritone sub, chromatic |
| 3 | Малотерцовое (m3↑/↓) | C → Eb or C → A | Parallel keys, relative |
| 4 | Восходящее целотоновое (M2↑) | C → D | Secondary dominants |
| 5 | Нисходящее большетерцовое (M3↓) | C → Ab | Chromatic mediant |
| 6 | Тритоновое (tritone) | C → F# | Substitution |
| 7 | Одноимённое мажор/минор (same root) | C → Cm | Modal interchange |

---

## Tritone Substitution

### Principle
- Замена доминанты (V7) на bII7
- Share 3rd and 7th enharmonically
- Example: G7 → Db7 (in C major)
  - G7: B-F (3rd-7th)
  - Db7: F-Cb (enharmonically B) (3rd-7th)

### Application
- **Кварто-квинтовое** and **полутоновое** движение взаимозаменяемы
- Схема: Dm7 → Db7 → CM (instead of Dm7 → G7 → CM)

## Block Analysis System (Блоковый анализ)

### Three Harmonic Blocks
- **①** = Tonic (T)
- **②** = Subdominant (S)
- **③** = Dominant (D)

### Usage
- Group chord progressions into functional blocks
- Each block = устойчивая функциональная единица
- Analyze jazz standards by marking blocks

### Prohodящие обороты (Passing Progressions)

#### Внутриблоковые (Within Block)
- Проходящие аккорды within one functional block
- Fундаментальные тоны move by ступеням **уменьшённого лада** (восходящий/нисходящий)

#### Междублоковые (Between Blocks)
- Between two functional blocks
- Whole-tone or semitone relationship
- Хроматически скользящие = дополнительная "раскраска"

## Polyaccords (Полиаккорды)

### Structure
- Аккорд из двух или более аккордов разных тональностей
- **Фундаментальная часть**: прима (бас) + септима
- **Надстройка**: мажорное/минорное трезвучие или обращение
- Квинта и терция могут быть пропущены

### Notation
- **D/C7** — верхний символ = надстройка, нижний = базисный аккорд
- Example: D major triad over C7 bass = C13#11

### Usage
- Чаще всего как **педальные гармонии** (pedal point chords)
- Creates complex, modern voicings

---

## Blues Harmony (Блюзовый лад)

### Fонизм доминантовых аккордов
- "Нормой" в блюзе = **диссонирующие аккорды с малой септимой**
- All I, IV, V chords = dominant 7ths (not maj7)
- Fонизм = особое отношение к диссонансу как необходимой фонической принадлежности

### Blues Scale
- 1-b3-4-b5-5-b7
- Example in C: C-Eb-F-Gb-G-Bb
- b3, b5, b7 = "blue notes"

---

## Quartal Harmony (Квартаккорды)

### Structure
- Stacked fourths instead of thirds
- Example: C-F-Bb-Eb (4 notes, all perfect 4ths)

### Character
- Ambiguous, modern sound
- No clear major/minor quality
- Good for static/meditative textures

---

## Parallel Keys (Параллельные тональности)

### Definition
- Тональности с одинаковым количеством ключевых знаков
- Общий безклавишный звукорядок
- Example: C major / A minor

### малотерцовое движение
- Connects главные функции параллельных тональностей
- C major: CM → FM → G7
- A minor: Am7 → Dm7 → Em7(E7)

---

## Mажоро-минорная хроматическая тональность

### Principle
- Расширение тональности за счёт аккордов одноимённого минора/мажора
- All 12 хроматических ступеней могут быть фундаментальными тонами
- Тритоновые замены расширяют центр и периферию

### Example in C major
- Borrowed from C minor: Fm, Ab, Bb7, Db, Eb
- Creates richer harmonic palette

## Interval Contour Table (Контурное двуголосие)

### Principle
- Систематизация всех интервалов хроматического ряда
- Указание возможных базисных септаккордов
- Практическое использование: определить интервал → найти аккорды

### Usage
- Неподвижное основание + движущаяся вершина
- Or: движущееся основание + неподвижная вершина
- Generates bass lines and melodies from intervallic structure

---

## Modal Interchange (Гармоническая смена мажор/минор)

### Type 7 Movement
- Движение на одноимённое мажор/минор (same root)
- Example: C → Cm or Cm → C
- Immediate shift in color/mood

---

## Exercise Pattern: Tritone Voicings + Walking Bass

### Left Hand
- Тритонаккорды формы A и B (3rd + 7th only)
- Example: Dm7 → play F+C, G7 → play B+F

### Right Hand
- Шагающий бас четвертями (walking bass in quarters)
- Root movement by one of 7 intervallic types

### Improvisation
- **Блюзовый лад** in right hand over changes
- Мотивно-тематические структуры: 1+1+2, 2+1+1, 2+2, 4

---

## Simple vs. Complex Harmony (Усложнение)

### Simplified
- Джазовые темы часто записаны с упрощённой гармонией
- Leaves room for arranger to enrich

### Усложнение (Enrichment)
- **а) Аккордовая вертикаль**: добавленные тоны, квартаккорды, полиаккорды
- **б) Гармонические обороты**: substitute chords, passing chords, tritone subs

---

## MIDI Application

### 7-Type Movement Generator
1. Input: starting chord
2. Select movement type (1-7)
3. Generate next chord root
4. Apply chord quality based on function

### Tritone Substitution Algorithm
1. Identify V7 chords in progression
2. Replace with bII7 (tritone away)
3. Verify 3rd/7th enharmonic equivalence
4. Choose based on desired bass motion (semitone down vs. fifth down)

### Block Analysis Tool
1. Analyze chord progression
2. Mark each chord as ① (T), ② (S), or ③ (D)
3. Identify проходящие обороты
4. Create REAPER markers for visual reference

### Polyaccord Builder
1. Input: bass note + chord type (e.g., C7)
2. Select upper structure (e.g., D major triad)
3. Combine: D/C7 = C13#11
4. Output MIDI voicing

### Blues Progression Template
- I7 (4 bars) - IV7 (2 bars) - I7 (2 bars) - V7 (1 bar) - IV7 (1 bar) - I7 (2 bars)
- All chords = dominant 7th type
- MIDI generator with blues scale overlays
